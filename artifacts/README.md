# Artifacts

This folder stores public outputs from IDEA loops.

Use it to collect:

- Screenshots
- Demo links
- Tweet/X links
- Notes and outcomes

## Suggested structure

Create one markdown file per loop:

- `YYYY-MM-DD-loop-name.md`

Example:

- `2026-02-15-onboarding-copy-test.md`

## Suggested entry format

```md
# Loop: <name>

## Goal
<what this loop aimed to test>

## Output
<links to prototype, code, screen, demo>

## What we learned
<key assumptions validated/invalidated>

## Next move
<ship / iterate / pivot / stop>

## Public links
- Tweet: <url>
- Thread: <url>
- Demo: <url>
```
